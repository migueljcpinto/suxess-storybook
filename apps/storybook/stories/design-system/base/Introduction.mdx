# Design System Base

Bem-vindo ao **Suxess Design System**! Esta é a coleção base de componentes reutilizáveis e acessíveis.

## 🎯 **Objetivo**

Fornecer uma base sólida de componentes UI que podem ser:
- **Reutilizados** em diferentes projetos
- **Customizados** para necessidades específicas
- **Acessíveis** por padrão
- **Consistentes** em design e comportamento

## 🏗️ **Estrutura**

### **Componentes Base**
- **UI Components**: Botões, inputs, modais, etc.
- **Layout Components**: Cards, separators, etc.
- **Navigation Components**: Menus, breadcrumbs, etc.

### **Clientes Específicos**
- **TKAG**: Componentes personalizados para o cliente TKAG
- **TKMS**: Componentes personalizados para o cliente TKMS

## 🎨 **Tecnologias**

- **React** + **TypeScript**
- **Tailwind CSS v4**
- **Radix UI** (acessibilidade)
- **Lucide React** (ícones)

## 🚀 **Como Usar**

1. **Componentes Base**: Use diretamente do `@repo/design-system`
2. **Clientes Específicos**: Explore as seções TKAG e TKMS
3. **Customização**: Adapte os componentes às suas necessidades
